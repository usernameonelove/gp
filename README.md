# Gym Progress PWA

Минималистичная программа для отслеживания прогресса в зале.

## Что есть

- Запись упражнения: вес, повторы, подходы, заметка
- История тренировок
- Статистика за неделю
- Лучшие результаты по упражнениям
- Цели на день, неделю, месяц, год
- Экспорт и импорт данных
- Работает без сервера
- Данные хранятся в браузере через localStorage

## Как открыть локально

1. Распакуй архив.
2. Открой файл `index.html` двойным кликом.

Важно: офлайн-режим через service worker работает только при запуске через сайт/локальный сервер, а не через file://.

## Как запустить локально правильно

Если установлен Python:

```bash
cd gym-progress-pwa
python -m http.server 8080
```

Потом открой:

```text
http://localhost:8080
```

## Как бесплатно открыть на iPhone

Самый простой вариант — GitHub Pages.

1. Создай репозиторий на GitHub.
2. Загрузи туда файлы из папки `gym-progress-pwa`.
3. Открой Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. Открой выданную ссылку на iPhone через Safari.
7. Нажми Поделиться → На экран «Домой».

После этого программа будет выглядеть почти как обычное приложение.

## Резервная копия

Во вкладке «Прогресс» нажми «Экспорт». Получишь JSON-файл со всеми данными.

Чтобы восстановить данные, нажми «Импорт» и выбери этот JSON-файл.
